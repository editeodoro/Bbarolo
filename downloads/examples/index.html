A couple of examples can be downloaded from here.

After having decompressed the archive, you can load the *.par files in the GUI and run BBarolo.
Otherwise, from the command line you can run the examples through:

> Bbarolo -p examples/n2403.par

The data cube of NGC2403 is from Fraternali et al. 2004.
