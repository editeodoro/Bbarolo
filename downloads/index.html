BBarolo is available for Mac OS X and Linux platforms. The code runs as a command line program only; since version 1.7, the Graphical User Interface (GUI) has been discontinued and will not be provided anymore as a pre-compiled binary.

Pre-compiled binaries are available here (additional libraries not needed, but still need Gnuplot and Astropy for the output plots).
